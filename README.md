# Simple Chess Game

A small, client-side chess demo built with `chess.js` and `chessboard.js`.

How to run locally

- Open a terminal and change to the project folder:

  cd "D:\AI-Projects\Chess"

- Serve the folder with Python's simple HTTP server:

```powershell
python -m http.server 8000
```

- Open the game in your browser at: http://localhost:8000

Notes

- This project is static (HTML/CSS/JS) and requires an HTTP server for some browsers to load scripts reliably.
- Files included: `index.html`, `script.js`, `style.css`, and `lib/` dependencies.
